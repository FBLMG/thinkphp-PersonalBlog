## 简介

个人博客系统以Thinkphp框架为基础，页面配合bootstrap为框架搭建。主要分为前后端功能

## 后端管理员

后台管理员包括以下功能呢个

*  首页基本信息查看
*  文章列表、文章的添加、文章的修改、文章的删除、评论查看
*  图片的查看、图片的发表、图片的修改、图片的删除
*  用户的管理、用户的查看、用户的编辑、用户的删除
*  用户留言的查看、用户留言的回复

## 前台用户

前台用户有以下功能

*  日记查看、日记的评论
*  图片的查看
*  留言回复与查看
*  个人中心的留言、评论


## 以下技术

*  短信接口
*  AJAX提交
*  图片无刷新上传
